# ヒーロー用 Lottie アニメーション

「Robot says hello」を表示するには、Lottie JSON をこのフォルダに配置してください。

1. [LottieFiles - Robot says hello](https://lottiefiles.com/free-animation/robot-says-hello-iUyDAnJFQ3) を開く
2. **Download** → **Lottie JSON** を選択してダウンロード
3. ダウンロードしたファイルを **`robot-hello.json`** にリネーム
4. このフォルダ（`public/animations/`）に保存

以上でヒーローセクションにロボットのアニメーションが表示されます。
